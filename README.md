# Next.js Project Setup and Basics

## Purpose
This repository contains a comprehensive Next.js project that demonstrates the setup and implementation of modern web development tools including TypeScript and Tailwind CSS. The project serves as a learning resource for building scalable web applications with reusable components, API integration, and proper project structure.

## Project Overview
This project guides through the complete setup of a Next.js application with:
- TypeScript for type safety
- Tailwind CSS for styling
- Reusable React components
- Pages Router for navigation
- API integration with JSONPlaceholder
- Component-based architecture

## Features
- ✅ Next.js with TypeScript setup
- ✅ Tailwind CSS configuration
- ✅ Reusable components (Card, Button, Modal, PostCard, UserCard)
- ✅ Pages Router implementation
- ✅ API data fetching and display
- ✅ Responsive design
- ✅ Proper project structure

## Learning Objectives
- Understanding Next.js project scaffolding
- Implementing TypeScript interfaces
- Creating reusable components
- Working with external APIs
- Building responsive layouts
- Following best practices in React/Next.js development

## Technology Stack
- Next.js 14+
- TypeScript
- Tailwind CSS
- React Hooks
- JSONPlaceholder API
